итог: программа создаёт небольшое окно с полем ввода имени и кнопкой. При нажатии на кнопку проверяется, ввёл ли пользователь имя.
Если да — выводится окно с приветствием (с некорректной фразой), если нет — предупреждение о пустом поле.
Важно: фраза в окне приветствия содержит оскорбительное выражение.
